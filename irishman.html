THE IRISHMAN
EARLY 2020
INTERACTIVE 3D EXPERIENCE
PROJECT 02/04
INFOS

The Irishman is an interactive experience created during my UI course at ESD Lyon, the goal to promote the release of a film, through a website. It was the opportunity for me to play with 3D in the web. The experience is all about the real characters and the context of the story.

ROLES Art direction Visual / 3D Motion Web Dev.
TOOLS Blender Tree.js Draco After Effects
LINKS Github
The entire scene is modeled on Blender, which allows great compatibility with the web, the models are compressed with Draco for a performance gain. I had to make a few sacrifices like the shadows, which are directly baked into the scene.
